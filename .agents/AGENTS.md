# Git Workflow Rule

Todas as alterações que forem testadas, validadas e confirmadas com sucesso devem ser imediatamente comitadas e enviadas (push) para o repositório git do projeto.
Repositório alvo: https://github.com/allanzinhodev/backlands_mmo

Certifique-se de perguntar ao usuário se ele deseja que você crie o commit e faça o push assim que uma feature ou correção for finalizada e validada.
